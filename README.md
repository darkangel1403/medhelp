============================== Системные требования ==============================

Данное программное средство «MedHelp» предназначено для автоматизации работы регистратуры городской поликлиники.
Основные функции: учет пациентов и врачей, электронная запись на прием, вызов врача на дом, ведение личных кабинетов, 
формирование отчетов и экспорт данных в различные форматы (Excel, CSV, JSON).

Для корректной работы программы необходимо:

1. Операционная система: Windows 10 / 11 (возможна работа на Windows 8.1 при наличии обновлений).
2. Платформа: .NET 8.0 Desktop Runtime (обязательно для запуска приложений на .NET 8).
   Ссылка для скачивания: https://dotnet.microsoft.com/download/dotnet/8.0
3. База данных: MS SQL Server LocalDB (обычно устанавливается вместе с Visual Studio или отдельным пакетом).
   Приложение автоматически пытается подключиться к экземпляру "(localdb)\MSSQLLocalDB".
4. Файлы программы: Распаковать содержимое архива в любую папку на компьютере.

ПОРЯДОК ЗАПУСКА:
1. Перейдите в папку «EXE».
2. Запустите файл «MedHelp.exe».
3. При первом запуске приложение проверит подключение к базе данных. 
   Если база не создана или недоступна, появится сообщение об ошибке.
   (Для создания БД используйте скрипт из папки PRG или обратитесь к администратору).

УЧЕТНЫЕ ДАННЫЕ ДЛЯ ВХОДА:

Вариант 1: Вход как АДМИНИСТРАТОР (полный доступ)
   Логин: admin
   Пароль: 1234
   Права: Управление справочниками врачей и пациентов, генерация расписания, просмотр всей статистики.

Вариант 2: Вход как ПАЦИЕНТ (пользователь)
   Для входа необходимо сначала зарегистрироваться.
   1. На главном экране нажмите кнопку «Регистрация нового пользователя».
   2. Заполните поля (ФИО, Логин, Телефон, Адрес, Пароль).
      - Логин: только латинские буквы и цифры.
      - ФИО: только русские буквы.
      - Телефон: формат +375XXXXXXXXX.
   3. После регистрации войдите под своим логином и паролем.
   Права: Запись к врачу, вызов на дом, личный кабинет, просмотр своей истории, экспорт своих данных.

ВОЗМОЖНОСТИ ПРОГРАММЫ:
- Поиск врачей и пациентов по фамилии (мгновенная фильтрация).
- Сортировка данных в таблицах по клику на заголовок столбца.
- Экспорт данных в файлы: Excel (.xlsx), CSV (.csv), JSON (.json), TXT (.txt).
- Печать электронных талонов на принтер.
- Отправка отчетов через почтовый клиент (кнопка «Отправить на почту»).
- Горячие клавиши:
  * Enter — подтверждение действия.
  * Esc — отмена / закрытие окна.
  * Ctrl+F — фокус на поле поиска.
  * F1 — окно «О программе».
  * F2 — окно «О разработчике».
  * F3 — окно «Справка».
  * Ctrl+S (в личном кабинете) — быстрый экспорт данных.

МИНИМАЛЬНЫЕ АППАРАТНЫЕ ТРЕБОВАНИЯ:
- Процессор: Intel Pentium IV или аналогичный AMD (частота от 1.5 ГГц).
- Оперативная память: не менее 2 ГБ (рекомендуется 4 ГБ).
- Свободное место на диске: не менее 500 МБ (для программы и временных файлов).
- Разрешение экрана: не менее 1280x768 точек.
- Устройства ввода: клавиатура, мышь.

ТЕХНИЧЕСКАЯ ПОДДЕРЖКА:
При возникновении ошибок подключения к базе данных убедитесь, что служба SQL Server LocalDB запущена.
