MedHelp

MedHelp — это приложение для управления медицинской клиникой, разработанное на платформе .NET с использованием WPF (Windows Presentation Foundation). Приложение предоставляет функционал для записи пациентов, управления приемами, работы с архивами и администрирования системы.

📋 Возможности
Аутентификация пользователей — безопасный вход в систему с хешированием паролей
Управление пациентами — создание и редактирование карточек пациентов
Запись на прием — планирование визитов к врачам
Домашние визиты — организация выездов врачей на дом
Архив данных — хранение и просмотр исторических записей
Администрирование — управление пользователями и настройками системы
Специалисты — каталог врачей и медицинских специалистов

🛠️ Технологии
.NET — платформа разработки
WPF — фреймворк для создания пользовательских интерфейсов
Material Design in XAML — библиотека компонентов интерфейса
SQL Server LocalDB — система управления базами данных
ADO.NET — доступ к данным

📦 Требования
Windows 10 или выше
.NET Desktop Runtime (последняя версия)
SQL Server LocalDB
Visual Studio 2019/2022 (для разработки)

🔐 Безопасность
Пароли пользователей хранятся в хешированном виде
Используется параметризованный SQL для защиты от SQL-инъекций
Ролевая модель доступа к функциям приложения

📝 Структура данных
Основные модели:
User — пользователь системы (логин, пароль, роль)
Patient — пациент (карта, ФИО, дата рождения, контакты)
Doctor — врач (специализация, кабинет)
Appointment — запись на прием (дата, время, статус)
HomeVisit — домашний визит (адрес, время, статус)
AppointmentStatus — статусы записей
